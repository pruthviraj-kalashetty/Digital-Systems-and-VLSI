# 06. Logic Gates

[![Stage](https://img.shields.io/badge/Digital--Design--and--VLSI--Fundamentals-blue.svg)](#)
[![Focus](https://img.shields.io/badge/Focus-Logic%20Gates-green.svg)](#)

This module introduces the fundamental logic gates that form the building blocks of digital electronics. It covers the operation, truth tables, Boolean expressions, symbols, and applications of basic and universal logic gates used in combinational circuits, Verilog HDL, RTL Design, FPGA Development, and VLSI Design.

---

## 🎯 Learning Objectives

By working through this module, you will be able to:

- Understand the function and operation of fundamental logic gates.
- Interpret logic gate symbols, truth tables, and Boolean expressions.
- Differentiate between basic, universal, and exclusive logic gates.
- Analyze simple digital circuits using logic gates.
- Build a strong foundation for combinational logic design and digital circuit implementation.

## 🛠️ Core Concepts Covered

### 1. Basic Logic Gates

Study the AND, OR, and NOT gates, which perform the fundamental logical operations used in digital circuit design.

### 2. Universal Logic Gates

Understand NAND and NOR gates, their universal properties, and how they can be used to implement any Boolean function.

### 3. Exclusive Logic Gates

Learn the operation of XOR and XNOR gates and their applications in arithmetic circuits, parity generation, error detection, and digital comparison.

### 4. Logic Gate Analysis

Analyze logic gate behavior using truth tables, Boolean expressions, and logic symbols to design and understand combinational digital circuits.
