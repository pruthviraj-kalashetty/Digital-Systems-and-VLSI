## 🚀 Getting Started

If you are new to studying number systems, it is highly recommended to follow this learning path:
1. Start with the **Binary System** to understand the absolute basics of digital systems.
2. Move on to **Octal** and **Hexadecimal** systems to see how we group binary data.
3. Finish with the **Number System Conversion** guide to master switching between all of them.

## 🛠️ Key Concepts Covered

* **Base/Radix:** The number of unique digits used to represent numbers.
* **Positional Notation:** How the position of a digit affects its actual value.
* **Conversion Algorithms:** Step-by-step methods like successive division and positional weights.
